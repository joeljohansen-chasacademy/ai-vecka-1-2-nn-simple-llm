Quadrant Classifier ml5.js + p5.js

Klicka i canvasen för att samla in data (UL/UR/LL/LR), träna modellen och låt den klassificera muspekarens position.

status: init…

Instruktion

  1. Välj etikett (UL/UR/LL/LR).
  2. Klicka i canvasen för att samla datapunkter.
  3. Byt etikett och repetera för alla fyra hörn.
  4. Klicka Träna → Starta/stoppa prediktion.

Tips: Försök ha ungefär lika många punkter i varje hörn.
